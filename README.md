erlmunk
=====

erlmunk is an Erlang Cnode wrapper around the Chipmunk2D physics engine.

[![Build Status](https://travis-ci.org/lrascao/erlmunk.svg?branch=develop)](https://travis-ci.org/lrascao/erlmunk)

Building
--------

```sh
$ make
```
